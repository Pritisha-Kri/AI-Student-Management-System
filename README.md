# AI-Student-Management-System
A full Student Management System with AI-assisted timetable generation, attendance analysis, performance tracking, and role-based dashboards.

Frontend: React.js, React Router DOM, Axios, Tailwind CSS
Backend: Node.js, Express.js
Database: PostgreSQL
Authentication: JWT, bcrypt.js
AI Assistance: Rule-based AI scoring, attendance analysis, performance analysis, student risk prediction
Tools: Git, GitHub, Postman, VS Code
Deployment: Vercel, Render/Railway, Supabase/Neon PostgreSQL
